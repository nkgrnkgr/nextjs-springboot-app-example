# Next.js + SpringBoot

## How to

### start frontend

`npm run dev`

### start backend

#### IDE
IDE start BackendServerApplication.kt

#### CLI
`backend-server/gradlew bootRun`


### Backend Build & Deploy
```bash
cd backend-server
`./gradlew build`
`java -jar build/libs/backend-server-0.0.1-SNAPSHOT.jar`
```
